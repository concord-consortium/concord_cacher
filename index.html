The LOOPS Do It Yourself microportal lets students and teachers anywhere build their own probeware activities which can then be run as Java webstart applications on MacOS X, Windows, and Linux computers The activities can be used with probeware from five different vendors: Data Harvest, Fourier, Pasco, ImagiWorks, and Vernier.

The LOOPS Do It Yourself microportal was developed using the Ruby on Rails web framework and uses Java applications developed at Concord Consortium for the TEEMSS2 project. Learner data network-persistence is provided by SAIL, the Scalable Architecture for Interactive Learning. Activity assembly, interactive components, and fine-grained persistence provided by Concord Consortium's OTrunk framework.

Mac OS X Web Start Fix: If you are using Java 1.5 on Mac OS X 10.4 or 10.5, you will need to run our Fix MacOS Java 1.5 Web Start Utility once on each computer before you launch any activities. Without the fix, activities that you have run before will freeze during download. Click here to run it now.
